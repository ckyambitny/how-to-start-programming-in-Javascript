What is HTML?

  • It's markup language for web pages
  • It's all about the structure
  • Browser translates HTML into DOM

WTF is DOM?

It's confusing at the beginning

For now, all we care about it that DOM provides interface to access elements and their properties with JavaScript and do whatever we want to them, like:

Things we can do to DOM element with JavaScript

  • Find them
  • Change its style
  • Remove them
  • Append other elements
  • Animate

My first page. Head

                <!DOCTYPE html>
                <html lang="en">
                <head>
                    <title>My First Page</title>
                    <meta charset="utf-8">
                    <link rel="stylesheet" href="./styles/styles.css">
                    <script type="text/javascript" src="./scripts/main.js">
                </head>

Body

                <body>
                    <div class="big-div">
                        <p id="big-par"> Hello World! </p>
                    </div>
                </body>
                </html>

Add some styling with CSS

  • Create a selector
  • Describe how described element should behave
  • Keep your styles in separate file

style.css

                .big-div {
                    width: 500px;
                }
                #big-par {
                    font-size: 30px;
                }

main.js

                window.onload = function () {
                    var text = "Hello Warsaw";
                    var paragraph = document.querySelector("#big-par");
                    paragraph.innerHTML = text;
                };
